          Find/Replace

          Lets use the link above as an example. We want to convert local links from .md to .html‚Ä¶

          in IDE:

          Find: (\[.*?\]\(\./.*?).md(\))  
Replace: $1.html$2

          using Javascript:

          str.replace(\(\[.*?\]\(\./.*?).md(\))\g, "$1.html$2");

          So, no matter the complexity, just write the match/replace as:

          str.replace(\        \g, "       ")
