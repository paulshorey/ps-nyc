Thoughts:

I like to use these assignments to challenge myself and try out new things. I liked this one.

I thought it was very easy, so added some complexity:

  • Only used ES5, no React, no jQuery,
  • Minimum steps to embed into a webpage (just add the script tag)
  • minimum steps for user to interact with it - instead of including an extra link to trigger the "window open", I wanted to see if I can do it on form-submit, automatically after passing validation
  • It should work on all browsers, including IE**
  • **although, I only tested this in Chrome, as I'm not actually getting paid for this
  • Also, multiple forms can be included on one page,
  • and each one does not conflict with the other
  • and the user can not access their functionality
  • Other than that I didn't add anything fancy and unnecessary

I'm not sure why I enjoyed building this so much. Maybe because...

  • 1) it's simple and takes me back to a simpler time of web development
  • 2) this is user-interaction at its purest - without extra layers of processing and algorithms - just handle events, and serve UI elements to the visitor
  • 3) and ofcourse, it's what I enjoy about the web - it connects people by handling clicks and opening windows and accessing servers, I'm connected to it all
