Open-source

npm i universal-common-scripts - JavaScript functions. Like underscore/lodash, but more for data manipulation, filtering, sorting. For browser or Node.js

npm i colorful-console-logger - Universal console.log replacement. Adds color, and integrates with cloud-logging like LogDNA. For browser or Node.js

FlySwatter.tech - Browser extension to auto-close unwanted popups, banners, cookie notifications, intrusive ads, newsletter promotions, etc. Anything that is CSS position:fixed, is fixed! Does not trigger a site's ad-blocker blocker. Finally, browse in peace.

Dev notes - Tutorials, recipes, tips for developers. Working on using a note-taking platform (Notion, GitBook) as a website CMS. Publish your project documentation or knowldgebase onto your primary domain instead of onto a separate subdomain - for improved SEO.

JS jobs - Quick proof of concept. Took 3 days to build, including the data-mining. Nice interface to search through mounds of content. Can be easily adapted to other sources like latest news, events, wikipedia, and research or legal articles. Allows user to search using Regular Expressions.
