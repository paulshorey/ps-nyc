Open-source

npm i universal-common-scripts - JavaScript functions. Like underscore/lodash, but more for data manipulation, filtering, sorting. For browser or Node.js

npm i colorful-console-logger - Universal console.log replacement. Adds color, and integrates with cloud-logging like LogDNA. For browser or Node.js

npm i horizontal-vertical-carousel - The carousels you see here. Built in React. Very simple to install, customize, and allow custom CSS. See non-react version on Codepen.

FlySwatter.tech - Browser extension to auto-close unwanted popups, banners, cookie notifications, intrusive ads, newsletter promotions, etc. Anything that is CSS position:fixed, is fixed! Does not trigger a site's ad-blocker blocker. Finally, browse in peace.

Dev notes - Tutorials, recipes, tips for developers. Few years of JavaScript programming notes. When I come accross an interesting issue, I try to document it for others, or for myself in the future. Now working on adding general useful tips, and converting to "how to" articles.

JS jobs - Interface for technical users to search through mounds of content. Search using Regular Expressions. Can be easily adapted to other types of content like news, events, and research or legal articles. Quick proof of concept. Took a week to build, including data-mining. Made with React. articles.
